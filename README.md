# LangChain Document Loader Experiments

This repository contains experiments with LangChain document loaders and PDF processing for building RAG systems.

## Features
- Load text documents
- Load PDF files
- Process documents using LangChain
- Store embeddings in Chroma vector database

## Project Structure
notebook/
    
    document.ipynb
    
    pdf_loader.ipynb

main.py

## Installation

uv venv
source .venv/bin/activate
uv pip install -r requirement.txt

## Run

python main.py
